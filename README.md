# GAME OF LIFE

### Requirements

To run this project you need JDK 8. Since JDK 11, JavaFX is available as a separate module.

### RUN
Clone repository
```
git clone https://github.com/patrykkk141/game-of-life.git
```

Go to folder
```
/game-of-life/target/
```

type
```
java -jar game-of-life-jar-with-dependencies.jar
```
